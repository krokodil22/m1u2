# m1u2

В репозитории 5 отдельных статических сервисов. После деплоя на GitHub Pages каждый доступен по адресу:

- `https://krokodil22.github.io/m1u2/copy/`
- `https://krokodil22.github.io/m1u2/wheel/`
- `https://krokodil22.github.io/m1u2/trunks/`
- `https://krokodil22.github.io/m1u2/draw/`
- `https://krokodil22.github.io/m1u2/find/`

## Публикация

Добавлен workflow `.github/workflows/pages.yml`, который публикует весь репозиторий как статический сайт в GitHub Pages.

Чтобы всё работало по путям вида `/m1u2/<folder>/`, ссылки и ресурсы в сервисах используются через относительные пути.
